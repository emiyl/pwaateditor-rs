# pwaateditor-rs

PWAATeditor is a simple editor for Phoenix Wright: Ace Attorney Trilogy save files, written in Rust using eframe and egui. This program is a successor to [emiyl/PWAATeditor](https://github.com/emiyl/PWAATeditor).

This has only been tested for Windows save files.

## Features

- Unlocking Chapters
- Changing in-game settings
- Setting custom resolutions
